## Run Locally

Clone the project

```bash
  git clone https://github.com/vivekdevkar123/MyBlog.git
```

Go to the project directory

```bash
  cd MyBlog
```

Install dependencies

```bash
    pip install requirements.txt
```

Start the server

```bash
  python3 manage.py runserver
```
